Check if your wireless card supports monitor mode for Aircrack-ng and Wireshark?

Hi guys, how are you all doing ?

This will be a fairly small article, that aims to help you figure out whether your Wifi card supports the monitor mode. It can be really disheartening when you install wireshark, hoping to capture packets but you end up capturing the packets your PC sent to the Internet. But don’t you worry guys, I got you covered!

What is monitor mode?

Before rushing to the solution, lets first understand what monitor mode is? Monitor mode allows all the computers to capture all the packets from the wireless network without being associated with an access point. In layman’s terms, it allows you to capture all the packets in the air, irrespective of whether you are connected to a particular WiFi or not.

Checking support for the monitor mode.

For checking this, we need to install iw. This will be helping us check our WiFi adapter capabilities.

Installing on Linux

I have installed it on debian and it worked out of the box

sudo apt-get install iw


on arch based distributions-

sudo pacman -S iw


Now in the command line, type-

sudo iw list


This will be giving you a long list of features supported by your network card like bit rates, supported interface modes frequencies, device supports, etc. But what we are concerned about is the supported interface modes. There you will be finding all the modes supported and if there is a monitor mode, congratulations, you are good to go!

Troubleshooting/ Things to be kept in mind

  • Even though wireless card supports the monitor mode, but you still have to turn that on. There are some settings that you need to tinker around with inorder to turn the managed mode on.
  • This command will produce any out put on the virtual box and the virtual box can’t really interact with the hardware of your PC. This is the screenshot of me running it on Kali Linux, which is running on the Virtual Box.
  • You have to run it as sudo, otherwise, you will bet an error like “ command not found ”.
  • Generally, 80% of the wireless card supports monitor mode. You don’t need to have $20 wireless card to use the monitor mode. If you are using a laptop, there are pretty high chances that your wireless card will have monitor mode, unless the manufactures try doing cost cutting by putting in some cheap wireless cards.
